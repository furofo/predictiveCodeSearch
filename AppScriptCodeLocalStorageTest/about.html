Home Page

The default/home page has two sections. These are copy buttons that copy predefined coes or codes linked from a google sheet; and a code lookup tool that looks up linked codes from a google sheet and diplays them with their definition and the name of the dictionary they came from displayed above. An example of google sheet in proper formate will be listed below.

Copy Button Selection:

This section (pictured above) consists of buttons like listed above, when the button is clicked a predefined code is copied to the clipboard so the user can paste it else where.

You can change these buttons from a linked google sheet. To do this, first name one of the sheets ButtonSetup (code lookup tool will ignore this as a result). You can also put the value ButtonSetup in the third column and second row of the sheet. Then you can put values Button Text, Copy Value and Sheet Name from left to right in the first row of the spreadsheet. (This row is optional but doing this will help with readiblity. )

Then for each button you want to change, use the Button Text column for the button label (text on the button), and the Copy Value column for the value that copies to your clip board when you click it. This will work for the first three rows which correspond to the three buttons on the home page.

If you want to delete a button, simply type DELETE in Copy Value Column, this will also work for all three buttons and remove button toolbar complelty if all three have values of DELETE.

Code Lookup Tool:

This section (pictured above) is the bulk of the tool. When a user types a code and hits enter it will search for that code from a google sheet that a user has linked on setup page. This google sheet must have format listed in setup section of this page.

A user can either use search term and hit enter, or click the term that shows up in predective text drop down. They can also use arrow keys to select predective text that shows up and hit enter to search. If a match is found, the results will display below with a heading showing the type of code it is and a further description below. Additionally, if there is a website or ost linked the text will highlight in red inidcating you can click on it to open the link to more info about the code in a new web-page.

Setup Page

In this section you can link your own google sheet to the tool to pull from as long as it in the right format. Put in the website url to google sheet you want to link in the input field and click link sheet button to update it. Sheet should be in the following format:

Google Sheet Format:
  • The first row is completly optional and can be used to name the columns
  • The first Column is the dictionary keys or search terms
  • The second Column is the key or term definition
  • The thir column is any website links associated with the term. This is optional and if present will highlight the definition in red when this term is hovered over, indicating it is clickable
  • Each worksheet in the linked google sheet is a separate dictionary, the name of each dictionary is either the sheet name, or a optional vaue put in the 4th column / 1st row, this is the value that displays in parenthesis when searching for the search term
  • You can also name a worksheet Button Setup as described earlier in the copy button section. This customizes button format on Home Page. The format for this sheet is different and is described above in Copy Button Section.

Troubleshooting

Home Page

The Home and Setup pages have some error handling functionality, here are solutions to common errors:

  • No google sheet linked! This error means there is no google sheet linked to the tool. To resolve, click on setup link of message or got to set up page in navbar. Then put in a shareable google link. You can get a shareable link by clicking share in top right corner of webpage with google sheet then click copy link button. For this to work you need to have editor access to sheet.
  • Google sheet did not connect. This means that the google sheet did not connect succesfully. The next section shows the google sheet that is currently linked. Make sure the url is a shareable url that you have editor access to and that the sheet follows formatting guidelines in setup section of this page. To resolve, correct url in setup page or put google sheet in proper format.

  • No google sheet linked!This error means nothing was put in input field when link sheet was submitted. Put a shareable google sheet link in the input field and click link sheet button to resolve.
  • Google sheet did not connect. This means that google sheet did not connect succesfully. The next section shows the google sheet that is currently linked. Make sure the url is a shareable url that you have editor access to and that sheet follows formatting guidelines in setup section of this page.
