The navigation Menu sits at top of page and allows you to go to different web pages of the tool, by clicking them, the current web page will be highlgihted in black. The navigation menu should look like image above. Next We will discuss each page in further detail.

Home Page

The default/ home page has two sections, copy buttons that copy predifined codes common used in a dept. And a code lookup tool that looks up common codes and displays them in dictionary.

Copy Button Selection:

This section (pictured above) consists of buttons like listed above, when the button is clicked a predfinied code is copied to the clipboard so the user can paste it else where.

Code Lookup Tool:

This section (pictured above) is the bulk of the tool. When a user types a code and hits enter it will search for that code in predefined dictionaries and either a predfiined google sheet, or one that a user can link to on setup page.

A user can either use search term and hit enter, or click the term that shows up in the predictive text. The arrow keys can also select predective text / search results that show up and then hit enter to search for selection. If a match is found, the results will display below with a heading showing the type of code it is and a further description below. Additionally if there is a website or ost linked the text will highlight in red indicating you can click on it to open the link to find more detail about the code in a new page.

Setup Page

In this section you can link your own google sheet to the tool to pull from as long as it in the right format. Put in the website url to google sheet you want to link in the input field and click link sheet button to update it. Sheet should be in the follwing format

Google Sheet Format:
  • First row is completly opitional and can be used to name columns
  • First Column are the dictionary keys/ search terms
  • Second Column are the term definitions
  • Third column are ost/ website links this is optional and if present will highlight the defnition in red when displays indicating is clickable
  • Each worksheet in the linked google sheet is a separte dictionary, the name of each dictionary is either the sheet name, or a optional vaue put in the 4th column/ 1st row, this is the value that displays in parenthesis when searching for search term

Troubleshooting

This tool relies in part on predefined dictionaries and in part on a predefined or user linked google sheet in google docs. If having issues with certain codes not showing up start by refreshing page.

If having further issues and the sheet is a custom one user linked, check that sheet is formatted properly and then relink page with url in setup page then refresh home page once more.
